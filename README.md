# Simple Image Viewer

Simple Image Viewer with python on window.

## Getting Started

1. install requirements

```
pip install -r requirements.txt
```

2. Run main.py

```
python .\main.py
```

## Supported Converter

- Grayscale
- Black and White
- Bw_dithering
- Four_color
- Sepia

[Pillow Modes](https://pillow.readthedocs.io/en/stable/handbook/concepts.html#concept-modes)
